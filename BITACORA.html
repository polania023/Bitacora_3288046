PROYECTO FINAL DE FUNDAMENTOS DE SOFTWARE

Software

Conjunto de programas y rutinas que permiten a un sistema informático realizar tareas específicas.

Sistema operativo

Programa principal que gestiona los recursos de hardware y provee servicios a otros programas.

Aplicación

Programa diseñado para realizar una tarea específica para el usuario.

Middleware

Capa de software que conecta diferentes aplicaciones o servicios para que trabajen en conjunto.

Driver (controlador)

Programa que permite al sistema operativo comunicarse con un dispositivo de hardware.

Firmware

Software grabado en la memoria interna de un dispositivo, que controla su funcionamiento básico.

Software libre

Programas que pueden usarse, estudiarse, modificarse y distribuirse libremente.

Software propietario

Programas cuyo código fuente no está disponible y están bajo licencia restringida.

Código fuente

Instrucciones escritas en un lenguaje de programación que forman un programa.

Código objeto

Traducción del código fuente a instrucciones comprensibles por la máquina.

Compilador

Programa que traduce código fuente a código ejecutable.

Intérprete

Software que ejecuta instrucciones directamente sin generar un archivo compilado.

Editor de texto

Aplicación para crear y modificar archivos de texto plano.

IDE (Entorno de Desarrollo Integrado)

Herramienta que reúne editor, compilador, depurador y otras funciones para programar.

Depurador

Programa que ayuda a encontrar y corregir errores en el software.

Repositorio

Almacén donde se guarda y gestiona código fuente y versiones de software.

Gestor de versiones

Herramienta que controla cambios y versiones en el código de un proyecto.

Parche (patch)

Actualización pequeña que corrige errores o vulnerabilidades en un software.

Actualización

Nueva versión que mejora funcionalidades o corrige fallos.

Suite ofimática

Conjunto de aplicaciones de oficina (procesador de textos, hojas de cálculo, presentaciones, etc.).

Procesador de texto

Programa para crear y editar documentos escritos.

Hoja de cálculo

Aplicación que gestiona datos en tablas y realiza cálculos.

Base de datos

Conjunto estructurado de datos gestionado por software.

SGBD (Sistema de Gestión de Bases de Datos)

Programa que permite crear, manipular y administrar bases de datos.

CRM (Customer Relationship Management)
